## Run Locally

```bash
git clone https://github.com/Prohar04/Prohar-Saha-.git
cd Prohar-Saha-
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Deploy

Push to GitHub and connect the repo on [vercel.com](https://vercel.com).
Set the project name to `proharsaha` to get `proharsaha.vercel.app`.

Before deploying, add these files to `/public/`:

- `resume.pdf` — your CV
- `og-image.png` — 1200×630px cover image for social sharing
- `images/bob.png` — replace with your own profile photo
